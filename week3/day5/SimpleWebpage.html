    Well Come MY Web Application

    Content Section

    This mini project is about creating a simple webpage with a form to collect user data. The form includes fields like Full Name, Email, Password, Age, Gender, Message, and a Terms & Conditions checkbox.

    Key Points

    All required fields must be filled.
    Email format is validated using JavaScript.
    Password must be at least 6 characters.
    Checkbox must be checked before submission.
    Shows error messages for invalid inputs and a success message on valid submission.

    This project demonstrates HTML structure, basic CSS styling, and JavaScript form validation to make forms interactive and user-friendly.
